Arch

Arch Linux is an independently developed, x86-64 general-purpose GNU/Linux distribution that strives to provide the latest stable versions of most software by following a rolling-release model. The default installation is a minimal base system, configured by the user to only add what is purposely required.

CentOs

CentOS (Community Enterprise Operating System) is a Linux distribution that attempts to provide a free, enterprise-class, community-supported computing platform functionally compatible with its upstream source, Red Hat Enterprise Linux (RHEL).

Debian

Debian GNU/Linux provides much more than just the OS - a wide range of application software is included too. It comes with a total of over 29000 packages, precompiled software bundled up in a nice format for easy installation on your machine.

Fedora

Fedora is a distribution of Linux which forms the base for RHEL, though it is developed by the Fedora Project. The Fedora Project's mission is to lead the advancement of Free and open source software and content as a collaborative community.
